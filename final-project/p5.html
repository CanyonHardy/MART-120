P5.Js Projects

This program was really intimadating at first, and unlike most things, it only got more and more intimidating! Learning how to use it by making drawins was fun and helpful, but as we advanced to greater tools this prgram could do, the more things could go wrong. I felt really afraid of this programming for a while, but the longer I used it, the less afraid I became. But that didnt stop the massive amount of bugs I always had with it. I think these bugs helped me to learn about creative problem solving though, and ultimatley made me a better coder. It's really fun to see the advancement of my work in this site, from my drawing at the start, to my silly game at the end!
